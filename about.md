# About This Blog

Hi, I'm Nihal — a Computer Science student with an interest in history. I created this blog as part of my Internet Technologies course to explore how static site generators and GitHub Pages work.

## Why History?

I've always found history fascinating. There's something about learning how past events connect to the present that makes you see the world differently. Each article on this blog covers a topic that I find particularly interesting, from ancient civilizations to modern events that shaped our world.

## How This Site Was Built

This website was built using **Jekyll**, a static site generator that takes Markdown files and turns them into a full website. It's hosted on **GitHub Pages**, which is a free hosting service provided by GitHub for static websites.

The theme used is **Minima**, which is the default Jekyll theme. It's simple and clean, which suits a blog format well.

## Tools Used

- **Jekyll** — static site generator
- **GitHub Pages** — hosting
- **Markdown** — for writing content
- **Git** — version control

Thanks for reading!
